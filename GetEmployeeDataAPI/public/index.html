The documentation!


There are four potential GET calls with this API. To retrieve information, append the base URL with one of the following:


  1. /api/employees
    • This retrieves all of the information for all of the employees
  2. /api/employees/lastName/[last_name]
    • This retrieves all of the information for all of the employees with the specified last_name
    • Replace only the value inside the square brackets
  3. /api/employees/edu/[edu_level]
    • This retrieves all of the information for all of the employees with the specified edu_level
    • Use underscores (_) instead of what normally would be spaces
    • Replace only the value inside the square brackets with one of the following values (make sure it is singular, not plural):
      1. bachelor
      2. associate
      3. master
      4. doctorate
      5. vocational
      6. high_school
      7. less_than_high_school
  4. /api/employees/co/[company_name]
    • This retrieves all of the information for all of the employees with the specified company_name
    • Replace only the value inside the square brackets
    • Use underscores (_) instead of what normally would be spaces
